## Rooms (002) — 各部屋本文

### 4F 宙 (Sora) — the sky that holds your breath
```
最上階。高さは、それだけで一つの贅沢になる。
札幌の街明かりと、その向こうの夜空を独り占めする邸宅。
「宙」は、開拓の果てに辿り着く、無限の余白です。
```
強調: **「宙」** (accent color)

### 3F 木 (Ki) — the warmth of living timber
```
北海道の森が呼吸している。
無垢材の床に裸足を預け、木目に光が滲む朝を迎える。
やわらかな時間が流れる「木」は、
"暮らすように"の本意を最も強く感じられる一邸。
```
強調: **「木」**

### 2F 石 (Ishi) — silence carved into stone
```
時を経た岩のように、揺るがぬ落ち着き。
重厚な石材と陰影が織りなす空間で、思考は深く、呼吸はゆっくりと長くなる。
「石」は、静寂を所有する者のための部屋です。
```
強調: **「石」**

### 1F 土 (Tsuchi) — the soil that begins everything
```
開拓の起点。大地から立ちのぼる温もりと、土壁が抱く深い静けさ。
素足で歩きたくなる土間と、火を囲む夜のためのリビング。
この建築のすべてが、ここ「土」から始まります。
```
強調: **「土」**

## Philosophy (003) — philosophy-body

```
KAIDOは、ホテルでありながら、暮らすように過ごせる場所。
1日数組だけを迎える、限定された棲家です。
1部屋を広くとり、空間的にも、暮らし的にも、たっぷりと余白をひらく。
札幌駅徒歩圏内にありながら、最上級の静寂を保つ。
そして全室にサウナを備え、整える時間そのものを、一邸の中に置きました。

かつて開拓使がこの土地を切り拓いたように、
現代のゲストが、日常のしがらみを切り拓き、本来の自分を取り戻す。
その入り口でありたいと、私たちは考えています。
```
